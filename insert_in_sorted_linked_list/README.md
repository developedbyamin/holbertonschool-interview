## Description

The program includes a function `insert_node` which is responsible for inserting a number into a sorted singly linked list. The linked list is represented using the `listint_t` structure. The main program (`0-main.c`) demonstrates the usage of this function by creating a linked list, inserting a number, and printing the updated list.
